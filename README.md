# WNTRMUTE

A thing.


## Getting Started

Clone the repository and then

```
cd wintermute
brew update
npm install
npm run dev
```

### Prerequisites

You will need [Node.js installed](https://nodejs.org/en/download/)


## Built With

* [React](https://reactjs.org/)
* [Webpack](https://webpack.js.org/)
* [React Typist](https://github.com/jstejada/react-typist)
